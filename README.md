# PHASER

## 本地预览

- 执行脚本 `npm run dev`
- 在浏览器上通过 `localhost:8088/xxx/index.html` 浏览页面。`xxx` 对应 `apps` 目录下的文件夹名称，如 `a20190919firstgame`
